# 🐍 Python Bootcamp – From Fundamentals to Production

Welcome to my **5-Day Intensive Python Bootcamp** repository.

This repository documents my journey from Python fundamentals to writing production-quality Python code with engineering best practices. Instead of simply solving coding problems, every exercise focuses on understanding *how Python works internally* and applying that knowledge to real-world software development.

## 🎯 Goal

Build a strong Python foundation required for:

* Artificial Intelligence
* Generative AI
* Machine Learning
* Backend Development
* Production Software Engineering

By the end of this bootcamp, I aim to write clean, maintainable, and scalable Python code while following professional development workflows.

---

## 📅 Bootcamp Roadmap

### ✅ Day 1 – How Python Really Works

* Python Execution Model
* Variables & Object References
* Mutable vs Immutable Objects
* Memory Model
* Stack vs Heap (Conceptual)
* Functions & Namespaces
* LEGB Scope
* Debugging with VS Code & `pdb`

### ⏳ Day 2 – Professional Python

* Advanced Functions
* `*args` and `**kwargs`
* Lambda Functions
* Closures
* Decorators
* Modules & Packages
* Virtual Environments

### ⏳ Day 3 – Production OOP

* Classes & Objects
* Inheritance vs Composition
* Magic Methods
* Data Classes
* Type Hints
* SOLID Principles

### ⏳ Day 4 – Production Python Features

* File Handling
* JSON
* Exception Handling
* Logging
* Context Managers
* Iterators
* Generators

### ⏳ Day 5 – Mini Production Project

Build a complete Python application using everything learned throughout the bootcamp.
